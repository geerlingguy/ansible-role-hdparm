# Ansible Role: hdparm

[![Build Status](https://travis-ci.com/geerlingguy/ansible-role-hdparm.svg?branch=master)](https://travis-ci.com/geerlingguy/ansible-role-hdparm)

Installs hdparm on Debian/Ubuntu Linux servers.

## Requirements

None.

## Role Variables

Available variables are listed below, along with default values (see `defaults/main.yml`):

    hdparm_manage_config: true

Whether this role should manage the hdparm config.

    hdparm_devices: []
      # - dev: /dev/sda
      #   options:
      #     - "spindown_time = 120"

Options to set per device. Only used if `hdparm_manage_config` is `true`.

## Dependencies

None.

## Example Playbook

    - hosts: all
      roles:
        - geerlingguy.hdparm
